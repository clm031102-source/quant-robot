# Local GUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first local GUI for research, backtesting, and report viewing.

**Architecture:** Add a dependency-light standard-library HTTP server under `src/quant_robot/gui/`, backed by isolated demo fixtures and the existing factor/research/backtest modules. Static frontend assets render six tool pages and call local JSON endpoints.

**Tech Stack:** Python standard library HTTP server, pandas-backed existing quant modules, HTML, CSS, vanilla JavaScript, unittest, Playwright CLI or equivalent browser verification.
